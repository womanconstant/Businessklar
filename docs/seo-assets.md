# SEO Social Preview Assets

## Required Asset

- `og-image.png`
- Path: `/og-image.png`

## Recommended Specifications

- Size: `1200x630`
- Format: PNG or JPG
- Content: BusinessKlar logo/title, short subtitle, and a clean German business setup theme

## Tags To Add After The Image Exists

Add these tags to the `<head>` of each public root HTML page after `/og-image.png` exists in the repository:

```html
<meta property="og:image" content="https://businessklar.de/og-image.png">
<meta property="og:image:alt" content="BusinessKlar – business setup guidance for Germany">
<meta name="twitter:image" content="https://businessklar.de/og-image.png">
```
